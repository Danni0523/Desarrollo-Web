Lenguajes utilizados para el desarrollo de aplicaciones web

los 6 lenguajes mas utilizados para elabroar aplicaciones llamativas son:

Estos lenguajes son los mas utilizados hasta la actualivad para mantener un dessarrollo acptable para una aplicacion web :

    |1. Python:

Es el lenguaje artificial Actualmente quien domine este lenguaje tendra un trabajo bien pagado ya que es necesario dominarlo para las aplicaciones web y big data este tiene un leguaje en ingles y es sencillo.

    |2. Lenguaje C:

Centrado en la programación de bajo nivel, este lenguaje tiene más de 50 años de existencia, lo que le ha permitido convertirse en uno de los lenguajes de programación más utilizados y con mayor presencia en el mercado.Su uso se centra principalmente en escribir aplicaciones para múltiples plataformas.

    |3. Java:

Es uno de los lenguajes de programación más utilizados, ya que fue desarrollado para crear programas para dispositivos móviles, posteriormente se convirtió en una buena alternativa para la creación de aplicaciones web.

    |4. LenguajeC++:

Es considerado uno de los lenguajes más simples y efectivos. Sus orígenes provienen del lenguaje C, ya que tienen una sintaxis muy similar; sin embargo, el lenguaje C++ está orientado a objetos.

    |5. Lenguaje C#:

Es un lenguaje el cual esta orientado a objetos el cual fue desarrollado por Microsoft es usado en videojuegos y el desarrollo de aplicaciones web para los sistemas operativos iOS y Android .

    |6. HTML5

El HTML5 es la última versión del estándar HTML que se utiliza para crear las páginas web que estás visitando, e incorpora algunas novedades interesantes. Una de las notables, es darle cobertura a la reproducción de contenido multimedia.

    |7. CSS.

Se le conoce como Las hojas de estilo son un recurso que existe desde que se usaba el antecesor del HTML: el SGLM, un lenguaje para crear documentos electrónicos.
