Utilidad de el desarrollo de aplicaciones web

¿Cuáles son los tipos de aplicaciones web?

Actualmente existen 5 tipos de aplicaciones web las cuales son encargadas de desarrollar un mejor énfasis en el sector tecnológico.

Los 5 diferentes tipos son:

    |1. Aplicación web estática:

Es un sitio de internet que no modifica su contenido para personalizarlo y adaptarlo a cada interacción Y esta no recibe actualizaciones constantemente.

    |2. aplicación web dinámicas:

Es técnicamente mucho más complicado. Utilizan una base de datos para cargar información y este contenido se actualiza cada vez que el usuario visita la aplicación web. Los lenguajes utilizados para el desarrollo son: PHP y ASP.

    |3. Comercio o Tienda virtual:

El desarrollo es similar al comercio móvil o al desarrollo del comercio electrónico. Ya que estos requieren de bases de datos y tener un registro de compras con el cual se solicita el uso de billeteras electrónicas.

    |4. Portales web:

Es uno de los cuales el usuario interactúa en una serie de apartados donde se encuentran foros quejas reclamos botones de redes sociales.

    |5. Aplicaciones web con gestor de contenido:

El gestor de contenidos es muy común que los usuarios están en una constante actualización de contenido el cual hace que sea mas eficiente un ejemplo acorde seria WordPress.
